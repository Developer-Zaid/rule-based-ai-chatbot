# Rule-Based AI Chatbot

## Overview

This project is a Rule-Based AI Chatbot developed as part of Week 1 of the Artificial Intelligence Internship Program at DecodeLabs.

The chatbot uses predefined rules and decision-making logic to respond to user inputs and simulate basic human interaction.

## Features

- Greeting responses
- Help command
- Date and time information
- Identity queries
- Thank you responses
- Exit commands
- Unknown input handling

## Technologies Used

- Python 3
- Control Flow
- Dictionaries
- Functions

## How to Run

```bash
python chatbot.py
```

## Sample Commands

- hello
- hi
- hey
- help
- name
- who are you
- date
- time
- thanks
- bye
